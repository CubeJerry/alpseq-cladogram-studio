# AlpSeq Cladogram Studio

**Original AlpSeq repository:** [kzeglinski/alpseq](https://github.com/kzeglinski/alpseq)

A simple browser-based companion for viewing and restyling AlpSeq cladograms.

## Input

Drop in a completed AlpSeq `.html` report to extract its cladograms directly.

The app also accepts Plotly JSON, coordinate CSV/TSV files, sequence tables and FASTA files. Files are processed locally in the browser and are not uploaded.

## Features

- Detect and select cladograms from an AlpSeq report
- Adjust figure width, height, margins and orientation
- Change branch, leaf, outline, text and background styling
- Resize labels and leaf markers
- Hide labels, legends, hover data or axes
- Export PNG, SVG, standalone HTML or Plotly JSON
- Preserve the original imported plot data
